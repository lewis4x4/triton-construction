## 3. STARTING A JOB (CLOCK IN)

### Method 1: From Dashboard Time Clock
1. On your dashboard, find the **Time Clock** card
2. Tap the dropdown: **"Select a job to clock into..."**
3. Select your job from the list
4. Tap the green **"CLOCK IN"** button

### Method 2: From Inside a Job
1. Open the job (tap View Job)
2. Go to the **Time** tab
3. Tap the red **"Sign In to Project"** button

### What Happens When You Clock In
- Timer starts counting
- Job status changes to "In Progress"
- Your location and time are recorded
- Dashboard shows "ON THE CLOCK" with running timer

## 5. INSTALLING DEVICES

### Viewing Devices
1. Tap the **Devices** tab
2. You'll see a list of all devices to install
3. Each device shows:
   - Device name
   - Category (camera, sensor, access control, etc.)
   - Status badge (pending, installed)
   - Edit and Delete buttons

### Installing a Device
1. Find the device in the list
2. Tap **"Edit"** on the device
3. The device wizard opens
4. Fill in required information:

**For Cameras:**
- Serial Number (scan QR or type)
- MAC Address
- Location (where it's mounted)
- Installation photos

**For Sensors (Door/Window, Motion, Glass Break):**
- Serial Number
- Zone Number
- Location
- Test result (pass/fail)

**For Access Control (Door Controllers, Keypads):**
- Serial Number
- MAC Address
- Door Name
- Programming confirmation

**For Thermostats:**
- Serial Number
- MAC Address
- Zone Name
- WiFi connected confirmation

**For Light Switches:**
- Serial Number
- Location
- Circuit confirmation

5. Take installation photo(s) if required
6. Tap **"Save"** or **"Complete Installation"**
7. Device status changes to ‚úÖ Installed

### Adding Unplanned Items (Upsells)
If you install equipment not on the original list:
1. Tap **"+ Add Unplanned Item (Upsell)"** button
2. Select the product from the catalog
3. Fill in installation details
4. This gets billed to the customer

### Viewing by Location
1. Tap the **Locations** tab
2. See devices grouped by area (Front Door, Back Office, etc.)
3. Each location shows:
   - Number of devices installed vs total
   - Progress bar
   - Status (Complete/In Progress)

## 8. COMPLETING THE CHECKOUT CHECKLIST

### When to Complete
Complete the checkout checklist **before leaving the job site** and **before getting signatures**.

### Opening the Checklist
1. Tap the **Checkout** tab
2. You'll see the **"Return Trip Checklist"**
3. Shows progress: "X of X items completed"

### Checklist Categories
The checklist has multiple sections:

**Site Cleanup (5 items)**
- ‚òê Remove all installation debris
- ‚òê Vacuum/clean work areas
- ‚òê Remove protective coverings
- ‚òê Restore furniture/fixtures
- ‚òê Dispose of packaging properly

**System Verification (items vary)**
- ‚òê All devices powered on
- ‚òê System tested and operational
- ‚òê Customer trained on use
- ‚òê Passwords documented
- ‚òê Access codes programmed

**Documentation (items vary)**
- ‚òê All serial numbers recorded
- ‚òê Photos uploaded
- ‚òê Time entries complete

### Completing Items
1. Read each checklist item
2. Complete the task
3. Tap the checkbox to mark it done ‚úÖ
4. Add notes if needed (optional)
5. Continue until 100% complete

### Finishing Checkout
When all items are checked:
1. The progress shows **"100%"**
2. Tap **"Complete Return Trip Checkout"** button
3. This confirms you've done all closeout tasks

## 9. GETTING SIGNATURES

### When to Get Signatures
Get signatures **after**:
- ‚úÖ All devices installed
- ‚úÖ System tested and working
- ‚úÖ Customer trained
- ‚úÖ Checkout checklist complete

### Two Signatures Required

**1. Customer Signature**
- The store manager, owner, or authorized contact
- Confirms they accept the completed work

**2. Technician Signature**
- Your signature
- Confirms you completed the work as described

### Capturing Signatures
1. Tap the **Signatures** tab
2. You'll see two cards: Customer Signature & Technician Signature

**For Customer:**
1. Tap **"Capture Signature"** under Customer Signature
2. Hand the device to the customer
3. They enter their name
4. They sign with their finger on the signature pad
5. Tap **"Save Signature"**

**For Technician:**
1. Tap **"Capture Signature"** under Technician Signature
2. Enter your name
3. Sign with your finger
4. Tap **"Save Signature"**

### After Signatures
Both cards will show:
- ‚úÖ **"Signed"** badge
- Signature image
- Name of signer
- Date and time signed

## 11. TROUBLESHOOTING

### "I can't see my jobs"
- Check you're logged into the correct account
- Pull down to refresh the page
- Check internet connection
- Contact dispatch if jobs are missing

### "Clock In button is disabled"
- You must select a job from the dropdown first
- Make sure you have jobs assigned for today

### "Device won't save"
- Check all required fields are filled
- Make sure serial number is valid format
- Check internet connection
- Try again or contact support

### "Photo won't upload"
- Check internet connection
- Make sure photo isn't too large
- Try taking a new photo
- Use WiFi if cellular is slow

### "Signature won't capture"
- Make sure you're drawing on the signature pad
- Draw slower and more deliberately
- Try rotating device to landscape mode
- Refresh the page and try again

### "I forgot to clock in"
- Clock in now - late is better than never
- Contact dispatch to adjust time if needed
- Document actual arrival time in notes

### "I clocked into wrong job"
- Clock out immediately
- Clock into correct job
- Contact dispatch to fix the error
